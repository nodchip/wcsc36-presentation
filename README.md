# WCSC36 特別講演 Slidev deck

第36回世界コンピュータ将棋選手権 1 日目お昼休憩 特別講演
「将棋 AI の概要と最新動向」の Slidev スライドです。

## Setup

```powershell
npm ci
```

## Live preview

```powershell
npm run dev
```

## Build

```powershell
npm run build
```

## Build for GitHub Pages

GitHub Actions では `npm run build:pages` を使って `--base "/<repo>/"` を自動設定します。

```powershell
npm run build:pages
```

## Export PDF

```powershell
npm run export
```

## Assets

画像や図は `public/assets/` に置きます。
